# Cтворення схем української вишивки засобами Java

Програма є інтерактивним додатком, створеним на базі Java Swing, який дозволяє користувачу створювати орнаменти хрестиком. Інтерфейс повністю оптимізовано для роботи з мишою та клавіатурою завдяки гарячим клавішам.

## 1. Інструкція для користувача
### Як запустити проєкт:

### Варіант 1: Через середовище розробки IntelliJ IDEA 
1. Склонуйте або завантажте репозиторій із кодом.
2. Відкрийте папку проєкту в IntelliJ IDEA.
3. Переконайтесь, що JDK підключено (версія 17 або вище).
4. Знайдіть файл `Main.java`, натисніть на зелений трикутник поруч із методом `main` або натисніть `Shift + F10`.

### Варіант 2: Через термінал / командний рядок 
```bash
javac Main.java EmbroideryCanvas.java
java Main
```

# 🧾 Повна розкладка гарячих клавіш (Hotkeys)

Для зручності (UX) у додатку вимкнено стандартне перехоплення фокусу кнопками панелі інструментів (`.setFocusable(false)`), що дозволяє керувати процесом малювання прямо з клавіатури.

| Гаряча клавіша | Функція / Дія        | Опис |
|----------------|----------------------|------|
| ↑, ↓, ←, →     | Навігація по сітці   | Переміщує курсор по клітинках полотна |
| Space          | Нанесення хрестика   | Малює хрестик поточного кольору |
| Ctrl + Z       | Скасувати дію (Undo) | Видаляє останню групу хрестиків |
| Ctrl + S       | Швидке збереження    | Експорт схеми у PNG |
| P              | Палітра кольорів     | Виклик JColorChooser |
| M              | Керування музикою    | Вмикає/вимикає аудіосупровід |
| C              | Очищення полотна     | Стирає всі хрестики (з підтвердженням) |
| ESC            | Скасування вибору    | Скидає рамку вибору фрагмента |

# Всі функції та можливості програми

## 1. Стартова анімація вишивання
При кожному запуску програми користувач бачить ефектний запуск системи:  
- Традиційний український візерунок вишивається самостійно в реальному часі.  
- Використовується покроковий `javax.swing.Timer`.  
- Малювання відбувається у випадковому порядку (`Collections.shuffle`), що імітує живий процес роботи майстра.

---

## 2. Малювання мишею (двокнопковий режим)
- **Ліва кнопка миші (ЛКМ):** малює хрестик обраного кольору на полотні. Використовується подія `mousePressed` для максимальної швидкості та відгуку.  
- **Права кнопка миші (ПКМ):** працює як гумка, стираючи хрестики у клітинці (як у звичайному режимі, так і симетрично).

---

## 3. Режими симетрії
Програма підтримує 4 режими геометричного відображення (вибір через `JComboBox`):  
- **Без симетрії** — малювання відбувається лише у клітинці кліку/курсора.  
- **По горизонталі** — хрестик дублюється на протилежну сторону відносно горизонтальної осі центру полотна.  
- **По вертикалі** — дзеркальне малювання відносно вертикальної осі центру.  
- **Чотиристороння симетрія** — один клік або натискання пробілу створює одразу 4 симетричні хрестики у всіх чотирьох чвертях робочої сітки.

---

## 4. Інструмент «Дублювання орнаменту»
Дозволяє автоматично масштабувати та зациклювати візерунки на всю ширину та висоту полотна:

- Користувач натискає кнопку **«Дублювати»**.  
- У діалоговому вікні вводить ширину та висоту базового фрагмента в клітинках.  
- Вікно зникає, а за курсором миші по полотну починає слідувати напівпрозора синя рамка заданого користувачем розміру.  
- Клік ЛКМ фіксує область. Програма зчитує всі хрестики всередині рамки та автоматично розмножує цей фрагмент «встик» по всій площі сітки 41x29 з урахуванням обраного режиму симетрії.  

---

## 5. Багаторівнева історія дій (Undo)
- Програма динамічно підраховує кількість хрестиків, згенерованих за один клік чи натискання пробілу.  
- Враховуються складні режими симетрії (від 1 до 4 елементів одночасно).  
- Кількість доданих елементів записується у список `undoHistory`.  
- При натисканні кнопки **«Назад»** або комбінації **Ctrl + Z** додаток видаляє саме ту кількість хрестиків, яка була створена за останній крок, забезпечуючи коректне скасування.  

---

## 6. Імпорт та експорт у PNG з точним збереженням координат
- **Збереження (Експорт):** рендерить та експортує поточне полотно у файл `image.png` за допомогою `ImageIO.write`.  
- **Відкриття (Розпізнавання схем):** програма реалізує алгоритм реверс-інжинірингу картинок:  
  - Автоматично сканує пікселі PNG.  
  - Визначає точний лівий верхній кут білого полотна.  
  - Обчислює зміщення (offset).  
  - Зчитує кольори у центрі кожної клітинки (`CELL_SIZE`).  
  - Відновлює векторні хрестики на інтерактивній робочій сітці.  

---

## 7. Фоновий аудіосупровід
- Інтегрована система Java Audio (`Clip`, `AudioInputStream`).  
- При натисканні кнопки **«Музика»** або клавіші **M** запускається пісня Лелеки "Ridnym".  
- Музика автоматично зациклюється (`LOOP_CONTINUOUSLY`), створюючи приємну атмосферу для роботи над орнаментами.
